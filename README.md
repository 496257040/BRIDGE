# BRIDGE
BRIDGE-​B​​lood-tumor ​​R​NA ​I​ntegrated ​D​ataset for checkpointer biomarker ​​G​​eneration and ​​E​valuation
这是我的本科大创项目所使用的代码库，旨在通过整合血液与肿瘤组织的单细胞RNA测序（scRNA-seq）数据，开发和评估用于免疫检查点治疗相关的生物标志物。本项目基于Python生态中的Scanpy工具栈，提供了一套可复现的单细胞数据分析流程。

BRIDGE 项目专注于利用公开及合作产生的单细胞转录组数据，通过系统的生物信息学分析流程，识别和验证在血液和肿瘤微环境中具有预测价值的免疫检查点相关基因特征。本代码库整理了从原始数据处理到最终可视化的完整分析步骤，旨在提高分析的透明度和可重复性。
